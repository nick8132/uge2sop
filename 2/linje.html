Linje:

Formler:

Linjens ligning:

f(x) = a · x + b


Find a:

a =
y2-y1 / x2-x1

Find b:

b = y1 – a · x1

Information:

Hvis man har 2 punkter til en funktion ( f(x) = ax + b ), (x1, y1) og (x2, y2), kan man bruge denne her udregning til at beregne a og b med:

a =
y2-y1 / x2-x1

b = y1 – a · x1

De kan så bruges i linjens ligning som er:

f(x) = a · x + b


Eksempel:

Som eksempel kan vi tage to punkter som (-4,2) og (5,5).

Vi udregner a:

a =
y2-y1 / x2-x1
=
5 - 2 / 5 - (-4)
=
5 - 2 / 5 + 4)
=
3 / 9
= 0,3333

Vi udregner b:

b = y1 – a · x1 = 2 - 0,3333 · (-4) = 3,3333

a og b indsættes i linjens ligning:

f(x) = 0,3333x + 3,3333
